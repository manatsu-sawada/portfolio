# Manatsu Sawada Portfolio

看護師経験を持つWeb学習者としての自己紹介・制作物・イラストをまとめたポートフォリオサイトです。

## Site

- Portfolio: https://manatsu-sawada.github.io/portfolio/
- GitHub: https://github.com/manatsu-sawada/portfolio

## About

医療現場での経験から、医療とITをつなぐ分野に関心を持ちました。現在はWeb制作、AIツール、英語を学びながら、実用的で人に寄り添うサービスづくりを目指しています。

## Features

- 日本語 / 英語の表示切り替え
- レスポンシブ対応
- Worksセクションの制作物カード
- Illustrationセクションのサムネイル表示と拡大表示
- Contactリンク

## Tech Stack

- HTML
- CSS
- JavaScript

## Editable Image Paths

以下の画像を差し替えることで、表示内容を更新できます。

- `images/Hero_image.png`
- `images/my-portfolio-thumbnail.png`
- `images/coffee-bean-thumbnail.png`
- `assets/images/idol/noronoro.PNG`
- `assets/images/idol/kou2_choucreme.PNG`
- `assets/images/anime/favorite_chara.PNG`

## Notes

GitHub Pagesで公開する静的サイトです。ビルドツールは使用していないため、`index.html`、`style.css`、`script.js` を編集すればそのまま反映できます。
